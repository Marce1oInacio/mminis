Como usar

          
Fluxo completo para adicionar produtos e artigos ao site mminis.

        


        

          
🏎️ Adicionar um produto novo

          

            

              
1

              

                
Abra este painel

                
Dê dois cliques no arquivo admin.html no seu computador. Ele abre no
                  navegador, sem precisar de internet.

              

            

            

              
2

              

                
Preencha o formulário

                
Nome, descrição, preço, plataforma e link do afiliado. Para a foto: ou faz upload
                  da imagem do produto, ou cola a URL da foto.

              

            

            

              
3

              

                
Clique em "Gerar código"

                
O código HTML do card aparece na tela. Clique em Copiar.

              

            

            

              
4

              

                
Cole no GitHub

                
Abra github.com/Marce1oInacio/mminis, clique no
                  index.html, depois no lápis ✏️. Encontre <!-- FIM PRODUTOS --> e cole
                  o código antes dessa linha.
                

              

            

            

              
5

              

                
Salve

                
Clique em Commit changes. Em ~30 segundos o produto aparece no
                  site.

              

            

          

        


        

          
📝 Adicionar um artigo novo

          

            

              
1

              

                
Escreva o artigo

                
Crie um novo arquivo artigo-nome.html no GitHub (botão Add file →
                  Create new file). Use o rascunho do arquivo artigo-01-onde-comprar-hot-wheels.md como
                  base.

              

            

            

              
2

              

                
Gere o card

                
Na aba "Artigos" deste painel, preencha o formulário e gere o código do card de
                  preview.

              

            

            

              
3

              

                
Cole no index.html

                
Mesmo processo dos produtos — cole antes de
                  <!-- FIM ARTIGOS -->. Na primeira publicação, apague também o bloco
                  article-empty.
                

              

            

          

        


        

          
💡 Dicas rápidas

          

            

              
✓

              

                
Foto do produto

                
Baixe a foto da Shopee/Amazon, salve na pasta do projeto e use o upload. Ou cole
                  a URL direto da loja — sem precisar baixar nada.

              

            

            

              
✓

              

                
Remover produto

                
No GitHub, edite o index.html e apague o bloco entre
                  <!-- PRODUTO --> e <!-- /PRODUTO --> correspondente.
                

              

            

            

              
✓

              

                
Atualizar preço

                
Edite o index.html no GitHub, encontre o nome do produto (Ctrl+F) e
                  altere só a linha do preço.
